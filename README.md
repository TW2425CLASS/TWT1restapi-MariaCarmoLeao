## Tecnologias e APIs utilizadas
- Node.js, Express
- MongoDB Atlas
- Postman
- JSON-Server 
- Render

## Instalação e configuração
1. Clonar o repositório  
2. Instalar dependências: `npm install`  
3. Criar um ficheiro `.env` com as variáveis:  
   - MONGO_URI=...  
   - PORT=3001

## Como executar localmente
- `node server.js`
